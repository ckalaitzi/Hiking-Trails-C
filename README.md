# ⛰️ Route Management System (Σύστημα Διαχείρισης Διαδρομών)

Αυτό το έργο είναι μια εφαρμογή γραμμών εντολών (C) σχεδιασμένη για την αποτελεσματική διαχείριση, αποθήκευση και αναζήτηση δεδομένων σχετικά με διαδρομές πεζοπορίας/ορειβασίας. Η εφαρμογή χειρίζεται δυναμικά τη μνήμη και βελτιστοποιεί την ταξινόμηση για γρήγορη ανάκτηση δεδομένων.

## 🌟 Βασικά Χαρακτηριστικά

* **Δυναμική Διαχείριση Μνήμης**: Χρήση `malloc` και `free` για τη δυναμική δέσμευση και απελευθέρωση του πίνακα διαδρομών, εξασφαλίζοντας την αποφυγή διαρροών μνήμης (memory leaks).
* **Βελτιστοποιημένη Απόδοση**: Η ταξινόμηση των δεδομένων (φθίνουσα κατά μήκος) γίνεται μόνο μία φορά αμέσως μετά τη φόρτωση, βελτιώνοντας την απόδοση των πολλαπλών λειτουργιών αναζήτησης.
* **Ευελιξία I/O**: Υποστηρίζει φόρτωση δεδομένων τόσο από την τυπική είσοδο όσο και από αρχεία, καθώς και αποθήκευση των αποτελεσμάτων σε αρχείο.
* **Αναζήτηση Πολλαπλών Κριτηρίων**: Παρέχει εντολές για αναζήτηση με βάση το μήκος, την τοποθεσία, το υψόμετρο, τον εκτιμώμενο χρόνο και το προφίλ κλίσης (ανηφορική/κατηφορική).

## ⚙️ Τεχνικές Προδιαγραφές

* **Γλώσσα Προγραμματισμού**: C
* **Πρότυπο**: C99/C11
* **Δομή Δεδομένων**: `struct route` (Δομή)

### Υπολογισμός Εκτιμώμενου Χρόνου

Ο εκτιμώμενος χρόνος διαδρομής (σε λεπτά) υπολογίζεται με βάση το μήκος και την υψομετρική ανάβαση/κατάβαση:

$$\text{Time} = \frac{\text{Length}}{65.0} + \frac{\text{Uphill} \text{ (or } \text{Downhill)}}{10.0}$$

## 📁 Δομή Αρχείων

| Αρχείο | Ρόλος | Περιγραφή |
| :--- | :--- | :--- |
| `main.c` | **Κύριο Πρόγραμμα** | Περιέχει τη συνάρτηση `main()` και τον βρόχο εντολών (`do-while`) για το μενού. Διαχειρίζεται την αρχική φόρτωση, την ταξινόμηση και την απελευθέρωση μνήμης. |
| `funcs.h` | **Header** | Ορισμός της `struct route` και όλες οι δηλώσεις (prototypes) των συναρτήσεων. |
| `funcs.c` | **Υλοποίηση** | Περιέχει τον κώδικα για όλες τις λειτουργίες I/O, τους υπολογισμούς χρόνου, την ταξινόμηση (`sortRoutesByLength`) και τις συναρτήσεις αναζήτησης. |

## 🚀 Οδηγίες Μεταγλώττισης και Εκτέλεσης

## 📜 Εντολές Χρήστη

Αφού εκτελέσετε την εφαρμογή, εισάγετε μία από τις παρακάτω εντολές ακολουθούμενη από `Enter`.

| Εντολή | Περιγραφή | Απαιτούμενη Είσοδος |
| :---: | :--- | :--- |
| `i` | Φόρτωση δεδομένων από την **τυπική είσοδο** (`stdin`). | Πλήθος διαδρομών, ακολουθούμενο από τα δεδομένα κάθε διαδρομής. |
| `f` | Φόρτωση δεδομένων από **αρχείο**. | Όνομα αρχείου. |
| `p` | Εκτύπωση **όλων** των διαδρομών (κανονική και αντίστροφη φορά). | Καμία. |
| `l` | Αναζήτηση διαδρομών με **μήκος** $\le L$. | Αριθμός $L$. |
| `s` | Αναζήτηση διαδρομών που **ξεκινούν ή τελειώνουν** σε μια συγκεκριμένη τοποθεσία. | Όνομα τοποθεσίας. |
| `u` | Αναζήτηση διαδρομών με **περισσότερη ανάβαση** (`uphill > downhill`). | Καμία. |
| `d` | Αναζήτηση διαδρομών με **περισσότερη κατάβαση** (`downhill > uphill`). | Καμία. |
| `h` | Αναζήτηση διαδρομών εντός εύρους υψομέτρου. | Δύο αριθμοί: Ελάχιστο Υψόμετρο (Min) και Μέγιστο Υψόμετρο (Max). |
| `t` | Αναζήτηση διαδρομών με **εκτιμώμενο χρόνο** $\le T$. | Αριθμός $T$ (σε λεπτά). |
| `o` | Αποθήκευση των ταξινομημένων διαδρομών σε **αρχείο** εξόδου. | Όνομα αρχείου. |
| `q` | Έξοδος και τερματισμός της εφαρμογής. | Καμία. |
